# Co-Doctor Frontend

Modern React frontend for Co-Doctor Medical Appointment System.

## Features
- User Authentication with JWT
- Doctor Dashboard
- Patient Booking System
- Responsive Design

## Tech Stack
- React 18
- Tailwind CSS
- Axios
- React Router
- Zustand

## Setup
```bash
cd co-doctor-frontend
npm install
npm run dev
```

## Pages
- `/login` - User login
- `/register` - User registration
- `/dashboard` - Doctor dashboard
- `/booking` - Patient booking
